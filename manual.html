URNDR

by FillMember

Directly draw on top of the model. Or around it. Or somewhere in between.

TIPS:

#1   On some shortcuts you can hold SHIFT to engage special functions.
#2   Use effects and change parameters while exporting GIF for cool effects.

  • b - brush
  • e - eraser
  • f - fade
  • d - zig zag brush
  • z - shake
  • w - smooth
  • / - random color

This is an open source project. The source code is hosted on GitHub

URNDR can't be possible without these libraries, references and communities: GifPaint Three.js Watch.js jQuery dat.GUI jsGIF Font Awesome
